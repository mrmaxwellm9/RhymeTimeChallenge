Instructions:
1. Select the game mode and duration, then press "Start Game" to begin.
2. When the game starts, a word will be displayed above the submission field.
3. You have the selected time to submit as many words as possible that rhyme with the given word.
4. To submit a word, press Enter or click the "Submit" button.
5. Each correct rhyme will trigger a chime sound, increase your score by one, and display the word next to the starting word.
6. Incorrect words will trigger a buzzer sound.
Additional Information:
- It's nearly impossible to guess every rhyme for most words, so don't get discouraged if you miss a lot.
- The word list is sourced from the CMU Pronouncing Dictionary, and the rhymes are generated using API Ninja's Rhyme API.
